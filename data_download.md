# Data Sources — Sherlock-Lung TCGA LUAD Analysis

**Abid Al Reza, PhD**  
Prepared for: Computational Scientist II interview, CGR/FNL/NCI (req4512)

All required data are public and open-access. No dbGaP authentication is required for the TCGA LUAD files used in this reproducible pipeline.

Download from the project root:

```bash
bash scripts/download_data.sh
```

To skip files that are already present:

```bash
bash scripts/download_data.sh --skip-existing
```

The download script calls a helper script for the somatic MAF file:

```text
scripts/download_luad_maf.py
```

That helper queries the GDC API for all open-access TCGA-LUAD masked somatic mutation MAF files, downloads them in batches, and concatenates them into one local compressed MAF file.

## Download Script Behavior

The main download script is:

```text
scripts/download_data.sh
```

It resolves the project root from the script location, unless `SHERLOCK_LUNG_BASE` is explicitly set. It downloads into:

```text
data/TCGA_LUAD/
```

The script performs four download steps:

1. Download and concatenate TCGA-LUAD somatic MAF files from GDC using `scripts/download_luad_maf.py`.
2. Download TCGA-LUAD HiSeqV2 expression from UCSC Xena.
3. Download TCGA-LUAD clinical matrix from UCSC Xena.
4. Optionally download PanCancer GISTIC2 thresholded CNA from UCSC Xena.

The MAF helper is called using an absolute project path:

```bash
python3 "$BASE/scripts/download_luad_maf.py" --output "$MAF_OUT" --batch-size 100
```

This avoids the path error that occurs if the main script changes into `data/TCGA_LUAD/` and then tries to call `scripts/download_luad_maf.py` relative to that data directory.

## Source Details

### 1. Somatic MAF — GDC API

- **Portal:** GDC Data Portal
- **Project:** TCGA-LUAD
- **Data type:** Masked Somatic Mutation
- **Access:** Open
- **Local output:** `TCGA_LUAD_somatic.maf.gz`
- **Downloader:** `scripts/download_luad_maf.py`

The helper queries:

```text
https://api.gdc.cancer.gov/files
```

using filters for:

```text
cases.project.project_id = TCGA-LUAD
data_type = Masked Somatic Mutation
access = open
```

It then downloads the returned file IDs in batches from:

```text
https://api.gdc.cancer.gov/data
```

The output is a single concatenated gzip-compressed MAF used by `scripts/02_MAF_analysis.R`.

### 2. Gene Expression — UCSC Xena

- **Portal:** UCSC Xena
- **Hub:** TCGA Hub
- **Dataset:** `TCGA.LUAD.sampleMap/HiSeqV2.gz`
- **Values:** normalized/log-transformed RNA-seq expression values used by UCSC Xena
- **Local file:** `TCGA_LUAD_HiSeqV2.gz`

Primary URL used by the script:

```text
https://tcga-xena-hub.s3.us-east-1.amazonaws.com/download/TCGA.LUAD.sampleMap%2FHiSeqV2.gz
```

Fallback URL:

```text
https://tcga.xenahubs.net/download/TCGA.LUAD.sampleMap/HiSeqV2.gz
```

### 3. Clinical Data — UCSC Xena

- **Portal:** UCSC Xena
- **Hub:** TCGA Hub
- **Dataset:** `TCGA.LUAD.sampleMap/LUAD_clinicalMatrix`
- **Local file:** `TCGA_LUAD_clinical.tsv`
- **Key variables used:**
  - `tobacco_smoking_history`
  - `vital_status`
  - `days_to_death`
  - `days_to_last_followup`

Smoking group definitions:

| TCGA code | Meaning | Group used here |
|---|---|---|
| 1 | Lifelong non-smoker | Never-smoker |
| 2 | Current smoker | Smoker |
| 3 | Current reformed smoker, >15 years | Smoker |
| 4 | Current reformed smoker, ≤15 years | Smoker |
| 5 or missing | Duration unknown / unavailable | Excluded from smoking-group comparison |

Primary URL used by the script:

```text
https://tcga-xena-hub.s3.us-east-1.amazonaws.com/download/TCGA.LUAD.sampleMap%2FLUAD_clinicalMatrix
```

Fallback URL:

```text
https://tcga.xenahubs.net/download/TCGA.LUAD.sampleMap/LUAD_clinicalMatrix
```

### 4. Copy Number Alterations — UCSC Xena, Optional

- **Portal:** UCSC Xena
- **Hub:** TCGA Hub
- **Dataset:** `TCGA.PANCAN.sampleMap/Gistic2_CopyNumber_Gistic2_all_thresholded.by_genes.gz`
- **Local file:** `TCGA_PANCAN_CNA.tsv.gz`
- **Values:** GISTIC2 thresholded gene-level calls
  - `-2` = deep deletion
  - `-1` = shallow deletion
  - `0` = neutral
  - `1` = low-level gain
  - `2` = high-level amplification

Primary URL used by the script:

```text
https://tcga-xena-hub.s3.us-east-1.amazonaws.com/download/TCGA.PANCAN.sampleMap%2FGistic2_CopyNumber_Gistic2_all_thresholded.by_genes.gz
```

Fallback URL:

```text
https://tcga.xenahubs.net/download/TCGA.PANCAN.sampleMap/Gistic2_CopyNumber_Gistic2_all_thresholded.by_genes.gz
```

The integrated script filters this PanCancer matrix to TCGA-LUAD samples using the first 12 characters of each TCGA barcode.

## Barcode Matching Strategy

TCGA files use different barcode lengths across modalities. This analysis matches samples by the first 12 characters, corresponding to the TCGA patient identifier.

| Data type | Example | Match key |
|---|---|---|
| Clinical `sampleID` | `TCGA-05-4244-01` | `TCGA-05-4244` |
| MAF `Tumor_Sample_Barcode` | `TCGA-05-4244-01A-11D-A24D-08` | `TCGA-05-4244` |
| Expression column | `TCGA-05-4244-01` | `TCGA-05-4244` |
| CNA column | `TCGA-05-4244-01` | `TCGA-05-4244` |

The scripts also normalize `.` to `-` because some R import functions convert TCGA barcodes into syntactic names.

## Expected Matched Sample Counts

Exact sample counts can differ slightly depending on whether the script counts clinical rows, unique patients, or modality-matched samples. The pipeline reports the final matched counts in the logs.

Typical counts observed in this project:

|---|---:|---:|
| Expression matched samples | 83 | 469 |
| MAF matched samples | 69 | 420 |

Do not hard-code these counts in the analysis. The scripts compute them from the downloaded files.

## Relevance to Sherlock-Lung

This project uses TCGA LUAD as an independent public comparison dataset. It is not the Sherlock-Lung cohort. The goal is to test whether public TCGA LUAD data show related patterns, including lower tobacco-associated mutational signal in never-smokers, EGFR enrichment, pathway differences, immune-score differences, and optional EGFR copy-number support.

## Troubleshooting

### GDC MAF download is slow

The MAF download queries many open-access per-sample files and downloads them in batches. This can take several minutes. Use `--skip-existing` after a successful run.

### CNA download fails

CNA is optional. If `TCGA_PANCAN_CNA.tsv.gz` is missing, the core analysis still runs. Only CNA-specific outputs are skipped.

### Telomere data

Telomere data are intentionally not included in this workflow. The tested LUAD-specific Xena telomere path was not available.
